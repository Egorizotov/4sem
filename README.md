Цель работы

Цель данной лабораторной работы — знакомство с инструментами построения пользовательских интерфейсов web-сайтов: HTML и CSS.

В ходе выполнения работы необходимо ознакомиться с реализацией простого калькулятора и выполнить задания по варианту.

Дополнительные задания
1. Визуальный эффект нажатия кнопок

Реализован эффект нажатия кнопок для повышения интерактивности интерфейса.

button:active { 
  transform: translateY(1px) scale(.99); 
}
2. Адаптивная сеточная раскладка кнопок

Реализована сеточная структура калькулятора с использованием CSS Grid, а также увеличение ширины отдельных кнопок.

#result + div > div {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 14px;
}

#btn_op_equal {
  background: #22c55e;
  color: #06210f;
  grid-column: span 2;
}

#btn_digit_0 { 
  grid-column: span 2; 
}
